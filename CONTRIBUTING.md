# Contributing to ExamCraft AI

First off, thank you for considering contributing to ExamCraft AI! It's people like you that make this a great tool for students.

## How Can I Contribute?

### Reporting Bugs
- Use the **Bug Report** template.
- Describe the expected vs actual behavior.
- Include screenshots if relevant.

### Suggesting Enhancements
- Use the **Feature Request** template.
- Explain why this feature would be useful for the platform.

### Pull Requests
1. Fork the repo and create your branch from `main`.
2. If you've added code that should be tested, add tests.
3. Ensure the test suite passes.
4. Make sure your code follows the existing style.
5. Write clear, human-sounding commit messages.

## Development Setup

1. Clone your fork.
2. `npm install`
3. `cp .env.example .env` and add your Groq key.
4. `npm run dev` to start the local server.

## Style Guide
- We use Tailwind CSS for styling.
- Keep components modular and reusable.
- Use functional components and hooks.

Happy coding!
